# salem-moderator
Town of Salem moderator for iPad

Salem Moderator Control Panel

A browser-based moderator control panel for running a Town of Salem–style card game with automated scoring and game flow assistance. Designed primarily for tablet use (iPad-friendly), this tool helps moderators manage players, track roles, and handle scoring efficiently during live games.

🔗 Project Link

View the project here:
https://github.com/thorsundal-ux/salem-moderator/blob/main/moderator_control_panel_ipad.html

⸻

📌 Features
	•	🎮 Game Management
	•	Track players and assigned roles
	•	Control game phases (day/night cycles)
	•	🧮 Automated Scoring System
	•	Calculates scores based on outcomes
	•	Reduces manual bookkeeping errors
	•	📱 Tablet Optimized UI
	•	Designed for iPad use
	•	Clean layout for quick moderator actions
	•	⚡ Lightweight & Fast
	•	Runs entirely in the browser
	•	No installation required


Getting Started

1. Clone the Repository

git clone https://github.com/thorsundal-ux/salem-moderator.git

2. Open the Control Panel

Simply open the HTML file in your browser:

moderator_control_panel_ipad.html

For best results, use:
	•	Safari (iPad)
	•	Chrome (desktop/tablet)

How to Use
	1.	Set up players and assign roles
	2.	Start the game and manage phases
	3.	Track eliminations and actions
	4.	Let the system calculate scores automatically
